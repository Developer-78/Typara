# ⌨️ Typara — Typing Speed Trainer

**[EN]** | [ES](#-typara--entrenador-de-velocidad-de-escritura) | [PT](#%EF%B8%8F-typara--treinador-de-velocidade-de-digitação)

---

A clean, fully featured typing speed trainer built with **pure vanilla JavaScript, HTML5, and CSS3** — zero frameworks, zero dependencies, single self-contained file.

> Built as a public portfolio project by A.M. | [Live Demo](https://Developer-78.github.io/typara)

---

## ✨ Features

- **Real-time WPM & accuracy tracking** — live feedback on every keystroke
- **Personal best detection** — tracks your record across sessions
- **4 difficulty levels** — Easy, Medium, Hard, Expert
- **5 typing modes** — Words, Sentences, Paragraphs, Numbers, and ✨ Stories
- **Stories mode** — 50 original inspirational passages across 3 languages with a no-repeat queue system
- **7 time settings** — 15s, 30s, 60s, 90s, 120s, 5m, 10m, and Unlimited
- **Keyboard mastery heatmap** — per-key speed and accuracy visualization
- **30-day practice activity tracker** — see your consistency over time
- **Performance analytics** — fastest/slowest characters, accuracy breakdown, consistency score
- **3 languages** — English, Spanish (ES), Portuguese (PT)
- **Settings panel** — dark/light mode, 3 cursor styles, 3 font sizes, 3 sound profiles
- **Real WAV keystroke sounds** — Typewriter, Soft, and Mechanical click styles
- **Fully offline** — no internet connection required after loading

---

## 🚀 Getting Started

No installation needed. Just open the file:

```bash
# Clone the repository
git clone https://github.com/Developer-78/typara.git

# Open in your browser
open typara.html
```

Or visit the **live demo** directly: `https://Developer-78.github.io/typara`

---

## 🎮 How to Use

1. Select your **time**, **difficulty**, and **mode**
2. Click **▶ Start**
3. Type the text shown — use **Backspace** to correct mistakes
4. When time runs out your results appear automatically
5. Check **My Stats** to track your progress over time

> ⌨️ **Desktop only** — Typara is designed for use with a physical keyboard.
